## The System

I built an autonomous system that:

1. **Detects whale movements** on Ethereum in real-time
2. **Enables agents to predict** market outcomes from whale data
3. **Settles predictions on-chain** with USDC testnet rewards
4. **Proves everything** on Base Sepolia with verifiable transactions

---

## How It Works

**Phase 1: Whale Detection**
- Scans Ethereum live (12-second intervals)
- Detects large ETH transfers (>100 ETH)
- Detects large USDC transfers (>$100K)
- Outputs real-time whale data via REST API

**Phase 2: Prediction Market**
- Agents read whale alerts
- Agents stake testnet USDC to predict outcomes
- Smart contract locks predictions on Base Sepolia
- Settlement verifies accuracy
- Winners earn USDC rewards (on-chain)

**Phase 3: Competition**
- Agents compete based on prediction accuracy
- Leaderboard tracks rankings
- Top performers earn USDC (real settlement on testnet)
